weibo crawler
===============================

version number: 0.0.1
author: Jferroal

Overview
--------

A python package that can be installed with pip.

Installation / Usage
--------------------

To install use pip:

    $ pip install weibo_crawler


Or clone the repo:

    $ git clone https://github.com/ezirmusitua/weibo_crawler.git
    $ python setup.py install
